# issue-board-tracking

> a GitHub App built with [probot](https://github.com/probot/probot) that adds issues with certain labels to project boards.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```

## Test

```sh
# run the test suite
npm test
```
